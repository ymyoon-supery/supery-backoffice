### Plan
- 스펙: .omc/specs/deep-interview-supery-backoffice.md (모호성 14.5%)
- 계획: .omc/plans/supery-backoffice-plan.md (Architect+Critic 3회 컨센서스)

### Stack
- Next.js 15 (App Router) + TypeScript + Tailwind + shadcn/ui
- Supabase (PostgreSQL + Auth + Storage + Realtime)
- Google Workspace APIs (Directory, Calendar, Drive, Chat)
- pdf-lib (서버리스 PDF), ExcelJS (Excel 출력)
- TanStack Query (Client Component 인터랙션)
- Vercel 배포 (2 Cron: Directory sync + Outbox processor)

### Key Architecture Decisions
- Hybrid RSC + Server Actions (Client Component에 TanStack Query 없이 useTransition 사용)
- SECURITY DEFINER + SET search_path = public, pg_temp 하드닝 (5개 함수 전체 적용)
- outbox_events 테이블 + claim_outbox_batch() RPC (FOR UPDATE SKIP LOCKED)
- CREATE UNIQUE INDEX uniq_attendance_daily ((recorded_at AT TIME ZONE 'Asia/Seoul')::date) — 인라인 UNIQUE 대신 표현식 인덱스
- leave_approval_steps / expense_approval_steps 분리 FK 테이블 (폴리모픽 FK 방지)
- validate_and_submit_leave() / approve_leave_step() / approve_expense_step() 원자적 RPC
- @radix-ui/react-badge 존재하지 않는 패키지 → 제거 (shadcn Badge는 커스텀 컴포넌트)
- CookieOptionsWithName 타입은 @supabase/ssr에서 import (strict 모드 호환)
- Supabase 조인 결과 타입 → as unknown as Type 이중 단언 필요 (배열 반환 특성)
